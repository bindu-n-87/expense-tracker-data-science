# Expense Tracker Data Science Project

## Overview
This project is a data-driven Expense Tracker that simulates real-world financial transactions using synthetic data. It provides insights into spending behavior, budgeting, and financial analysis using Python and Streamlit.

---

## Features
- Synthetic expense data generation
- Data cleaning and preprocessing
- Feature engineering (budget, spending level, expense type)
- Exploratory Data Analysis (EDA)
- Interactive Streamlit dashboard
- Budget tracking system
- Savings analysis
- Spending anomaly detection

---

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit

---

## Project Workflow
Data Generation → Cleaning → Feature Engineering → Analysis → Visualization → Dashboard

---

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run Streamlit app
streamlit run app.py
